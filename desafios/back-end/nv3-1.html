Desafio: Programa de Controle de Estoque Mercado

Nível 3 - Freelancer Iniciante

Requisitos: Conhecimento básico em um framework web e banco de dados

Um pequeno mercado do seu bairro sempre fez a gestão dos seus produtos usando papel e caneta

Visando automatizar e deixar esse processo mais eficiente, eles te contrataram para desenvolver um sistema simples.

Demanda:

Faça um CRUD (Criar, ler, atualizar e deletar) de produtos de supermercado, considerando que:

  • Cada produto possui um nome, preço e uma quantidade em estoque
  • Seja possível possível listar todos os produtos de uma categoria

Dicas:

  • Caso queira exibir o resultado em uma página, você pode utilizar um framework como o bootstrap para facilitar a estilização para que você foque apenas no back-end;
  • Utilize nomes com significado. Não utilize nomes genéricos como x, y ou abreviações como dis, temp, etc;
  • Tente dividir o problema em partes menores antes resolvê-lo.

Critérios de avaliação:

  • Solução do problema (Se o código funciona e atende ao problema)
    • 0-40 pts
  • Legibilidade (Facilidade de entender o código)
    • 0-30 pts
  • Desacoplamento (O quanto as partes do código são independentes)
    • 0-15 pts
  • Testes unitários (Se os testes foram bem escritos)
    • 0-10 pts
  • Nome das variáveis (Qualidade dos nomes utilizados)
    • 0-5 pts
